Exciting upcoming games

Diablo IV

  • An isometric action role-playing game set in a vast open world medieval apocalypse. The world will be much larger than in previous Diablo games.
  • This will be the first game in the series where you can customise your character! The game will feature five classes at launch: Barbarian, Druid, Sorceress, Rogue, and Necromancer.
  • Expected to release on June 6th 2023

Baldur's Gate 3

  • It is the third instalment in the Baldur's Gate series. It is an isometric RPG with turn-based combat where you adventure through the forgoteen realms.
  • It will be based on the rule set from the fifth edition of the Dungeons and Dragons rulebook.You can play alone or in a party of up to four in multiplayer.
  • It is set to release in August 2023

Warhammer 40K Rogue Trader

  • A role playing game set in the 41st illenium.
  • It is loosely based on the tabletop RPG set in the 40k universe. You will adventure as a space privateer and explore the Kroronus Expanse.
  • There is no official release date yet, but there is speculation that a release could happen in Q3 2023
